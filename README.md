# PetsHome 🐾

A responsive pet adoption web application built as part of a final school project. PetsHome is a modern platform that allows users to browse adoptable pets and enables admins to manage pet listings through a clean, family-friendly interface.

![PetsHome Screenshot](https://petshome-resit.netlify.app/)

## 📌 Project Goal

The goal of this project is to demonstrate the developer's capabilities in planning, designing, and implementing a fully functional front-end application using modern web development tools and techniques. It reflects both technical and visual development skills acquired over the past two years at Noroff.

---

## 📋 Project Brief

A pet adoption organisation has commissioned the development of a responsive web application for pet adoption. The platform allows users to browse adoptable pets, view detailed information, and allows admins to manage pet listings.

The application consists of two main user flows:

- **Customer-Facing Interface**  
  Users can browse adoptable pets, search, filter, and view detailed pet information.

- **Admin Interface**  
  Admins can register, log in, and create, edit, or delete pet listings.

---

## 🧩 Features (User Stories)

### General Users

- 🐾 View a grid of available pets
- 🔍 Search for a pet by name, breed, species or colour
- 🐶 Filter pets by species (Dogs, Cats, Other)
- 🐱 View detailed information about a specific pet
- 🔗 Share a pet's URL via clipboard copy

### Admins (Registered users with `@stud.noroff.no`)

- 📝 Register as an admin
- 🔐 Log in securely
- ➕ Add new pets to the adoption list
- ✏️ Edit existing pet listings
- ❌ Delete pets from the adoption list

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/ziconstr/petshome.git
cd petshome
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```
